# UI / Components

The frontend is built from reusable components and shared UI primitives.

## Component structure

- `src/components` contains page-specific and shared component groups.
- `src/components/common` houses shared layout components.
- `src/components/ui` is for reusable form controls, cards, and buttons.
- `src/app` contains route-level page logic and layout wrappers.

## Reusable components

Key reusable components include:

- buttons and input controls
- product cards and listing tiles
- modals and notifications
- page sections and content layouts

## Design system

The UI follows a minimal but polished design system:

- consistent spacing and typography
- soft shadows and rounded corners
- dark theme with high contrast text
- mono font for code and data blocks
- hover and transition effects for a premium feel

Reusable components are designed so developers can build new pages without duplicating styles or logic.
