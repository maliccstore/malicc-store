# API Reference

The API Reference is automatically generated from the project TypeScript sources using TypeDoc.

## What is included

- `src/services` API clients and business logic
- `src/store` slices, thunks, and state models
- `src/utils` helper functions
- `src/types` type definitions and interfaces

## How to use it

Use the generated docs to:

- inspect function signatures
- verify returned data shapes
- discover available service methods
- understand the public interfaces exposed by the frontend
