# Malicc Store Documentation

Malicc Store is a modern frontend storefront built to demonstrate scalable e-commerce architecture, developer-friendly state management, and an auto-generated TypeScript API reference.

## What is Malicc Store

Malicc Store is a frontend repository for a shopping experience with:

- a product catalog and search experience
- cart and checkout flows
- centralized Redux state management
- payment and order processing logic
- generated TypeDoc API documentation for key services

## Why it exists

This project exists to:

- provide a complete example of a production-looking React/Next app
- show how business logic and UI can be separated cleanly
- document frontend architecture for developer onboarding
- surface typed API contracts and state logic for maintainers

## Key concepts

- **Components**: reusable UI pieces in `src/components`
- **Services**: API interaction and business rules in `src/services`
- **Store**: Redux slices and thunks in `src/store`
- **Types**: strong TypeScript models in `src/types`
- **Docs**: auto-generated API docs in `docs/api`
